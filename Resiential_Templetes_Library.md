

# Civil Construction Estimation App

# Residential Construction Templates Library

Version 1.0

---

# Purpose

Provide pre-configured residential construction templates commonly used in Indian RCC residential buildings.

These templates are intended for:

* Quick Estimation
* Plan Based Estimation
* Preliminary BOQ Generation

Final dimensions and reinforcement shall always be verified by a qualified Structural Engineer.

# TEMPLATE USAGE RULE

Whenever user creates a project:

App shall suggest:

* Footing Template
* Column Template
* Beam Template
* Slab Template
* Staircase Template
* Wall Template
* Door Template
* Window Template

based on:

* Number of Floors
* Structure Type
* Quality Level

User can accept or modify templates.
