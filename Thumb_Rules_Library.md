
# Civil Construction Estimation App

# Quick Estimate Thumb Rules Library

Version 1.0

---

# Purpose

This library provides quick estimation rules for residential RCC buildings when detailed structural information is not available.

Applicable For:

* Residential Buildings
* Ground Floor Houses
* G+1 Buildings
* G+2 Buildings
* G+3 Buildings

Quick Estimate results are indicative only.

Detailed Estimate Mode should be used for final BOQ generation.

---

# QUICK ESTIMATE INPUTS

User Provides:

* Built-up Area
* Number of Floors
* Quality Level
* Structure Type

App Generates:

* Project Cost
* Cement Quantity
* Steel Quantity
* Sand Quantity
* Aggregate Quantity
* Brick/AAC Quantity

# APP USAGE RULE

Quick Estimate Mode shall use:

1. Built-up Area

* Material Summary
* Project Cost
* Cost Breakdown
* Procurement Estimate

without requiring structural drawings.
